🧠 Pengantar Generic
Fajar Rio Dinata | 251410025

Apa itu Generic? Fitur untuk membuat kode yang bekerja dengan berbagai tipe data secara type-safe.

📦 Analog: Kotak penyimpanan yang bisa menampung apa saja, tapi dengan label tipe.

✨ Manfaat:

  • ✔️ Type safety (keamanan tipe)
  • ♻️ Code reusability (penggunaan ulang)
  • 🔍 Compile-time checking
Slide 3: Generic Dasar - Fajar Rio Dinata (251410025)
⚠️ Masalah Tanpa Generic
Fajar Rio Dinata | 251410025
class Box {
  dynamic data; // Bisa apa saja
  Box(this.data);
}

void main() {
  var box = Box(42);
  int value = box.data; // Runtime error jika bukan int
  // Tidak aman!
}

❌ Tanpa generic, rawan runtime error karena type tidak diperiksa saat kompilasi.

📐 Sintaks Generic Dasar
Fajar Rio Dinata | 251410025

Type parameter: <T>, <E>, <K, V>

Konvensi: T = Type, E = Element, K = Key, V = Value, R = Return type

class Box {
  T data;              // <- tambah spasi indentasi (biasanya 2 spasi)
  Box(this.data);
}

void main() {
  var intBox = Box(42);        // T = int
  var stringBox = Box('Hello'); // T = String

  int value = intBox.data;          // Aman, pasti int
  // stringBox.data = 123;          // ERROR compile-time
}
Slide 5: Sintaks - Fajar Rio Dinata (251410025)
📦 Contoh Kelas Box<T>
Fajar Rio Dinata | 251410025
class Box<T> {
  T? _value;
  Box(this._value);
  T? get value => _value;
  void setValue(T value) { _value = value; }
}

void main() {
  var intBox = Box<int>(10);
  print('Int Box: ${intBox.value}'); // 10
  intBox.setValue(20);
  var stringBox = Box<String>('Hello, Generic!');
  print(stringBox.value);
}

✅ Generic membuat kelas dapat digunakan untuk berbagai tipe dengan aman.

Slide 6: Box - Fajar Rio Dinata (251410025)
🔗 Menggabungkan Tipe Berbeda
Fajar Rio Dinata | 251410025
String combine<T, U>(T value1, U value2) {
  return '$value1, $value2';
}

void main() {
  var result1 = combine<String, int>("Number is ", 10);
  print(result1); // Number is , 10
  var result2 = combine<int, double>(10, 20.5);
  print(result2); // 10, 20.5
}

🔥 Multiple type parameters memberikan fleksibilitas tinggi.

🔒 Generic dengan extends (Constraint)
Fajar Rio Dinata | 251410025
class NumberBox<T extends num> {
  T value;
  NumberBox(this.value);
  T getValue() => value;
}

void main() {
  var intBox = NumberBox<int>(10);
  var doubleBox = NumberBox<double>(3.14);
  // var stringBox = NumberBox<String>("test"); // Error! Bukan turunan num
}

🎯 extends membatasi tipe yang diperbolehkan (hanya turunan num).

Slide 8: Constraint - Fajar Rio Dinata (251410025)
🧩 Type Inference (Inferensi Tipe)
Fajar Rio Dinata | 251410025
T myFunc<T>(T value) => value;

void main() {
  var intValue = myFunc(10); // Dart infer T sebagai int
  print(intValue.runtimeType); // int
  var stringValue = myFunc("test");
  print(stringValue.runtimeType); // String
}

✨ Dart secara cerdas menebak tipe generic dari argumen.

🐾 Abstract Class & Generic Constraint
Fajar Rio Dinata | 251410025
abstract class Animal { void makeSound(); }
class Dog implements Animal { void makeSound() => print('Woof!'); }
class Cat implements Animal { void makeSound() => print('Meow!'); }
class AnimalCage<T extends Animal> {
  T animal;
  AnimalCage(this.animal);
  void sound() => animal.makeSound();
}

void main() {
  var dogCage = AnimalCage<Dog>(Dog());
  dogCage.sound(); // Woof!
  // var stringCage = AnimalCage<String>("test"); // Error
}

Slide 14: Perbandingan - Fajar Rio Dinata (251410025)
🗄️ Cache Generic (Contoh Lanjutan)
Fajar Rio Dinata | 251410025
class Cache<T> {
  final Map<String, T> _storage = {};
  void set(String key, T value) => _storage[key] = value;
  T? get(String key) => _storage[key];
  void remove(String key) => _storage.remove(key);
}

void main() {
  var userCache = Cache<String>();
  userCache.set('name', 'Alice');
  var productCache = Cache<Product>();
  productCache.set('p1', Product('Laptop', 1000));
}

💾 Reusable storage dengan keamanan tipe penuh.
