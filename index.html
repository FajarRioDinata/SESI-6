⚠️ Masalah Tanpa Generic

Slide 3

Tanpa generic, kita menggunakan dynamic yang rawan runtime error.

class Box {
  dynamic data;
  Box(this.data);
}

void main() {
  var box = Box(42);
  int value = box.data; // Runtime error jika bukan int ❌
}
📌 Masalah: Tidak aman karena tipe data tidak dicek saat kompilasi.

📖 Sintaks Generic Dasar

Slide 5

Type parameter: <T>, <E>, <K, V>

  • T = Type
  • E = Element (collection)
  • K, V = Key, Value
  • R = Return type
class MyClass<T> {
  T data;
  MyClass(this.data);
  T getData() => data;
}

T myFunc<T>(T value) => value;

void main() {
  var myIntClass = MyClass<int>(10);
  var myStringClass = MyClass<String>("test");
  print(myIntClass.getData());   // 10
  print(myStringClass.getData()); // test

  print(myFunc<int>(100));       // 100
  print(myFunc<String>("hello")); // hello
}

📦 Contoh Kelas Box<T>

Slide 6
class Box<T> {
  T? _value;
  Box(this._value);
  T? get value => _value;
  void setValue(T value) { _value = value; }
}

void main() {
  var intBox = Box<int>(10);
  print('Int Box Value: ${intBox.value}');   // 10
  intBox.setValue(20);
  print('After Set: ${intBox.value}');        // 20

  var stringBox = Box<String>('Hello, Generic!');
  print('String Box: ${stringBox.value}');
  stringBox.setValue('Hello World');
  print('Updated: ${stringBox.value}');
}

🔁 Multiple Type Parameters: Pair<K, V>

Slide 7
class Pair<K, V> {
  K key;
  V value;
  Pair(this.key, this.value);
  void printPair() => print('Key: \$key, Value: \$value');
}

void main() {
  var pair1 = Pair<String, int>('age', 30);
  pair1.printPair(); // Key: age, Value: 30

  var pair2 = Pair<String, String>('name', 'Alice');
  pair2.printPair();

  var pair3 = Pair<int, double>(1, 2.71);
  pair3.printPair();
}

🔧 Fungsi Generic: Combine<T, U>

Slide 8
String combine<T, U>(T value1, U value2) {
  return '\$value1, \$value2';
}

void main() {
  var result1 = combine<String, int>('Number is ', 10);
  print(result1); // Number is , 10

  var result2 = combine<int, double>(10, 20.5);
  print(result2); // 10, 20.5
}

🔒 Generic dengan extends (Constraints)

Slide 9

Membatasi tipe data dengan extends.

class NumberBox<T extends num> {
  T value;
  NumberBox(this.value);
  T getValue() => value;
}

void main() {
  var intBox = NumberBox<int>(10);
  print(intBox.getValue());   // 10
  var doubleBox = NumberBox<double>(3.14);
  print(doubleBox.getValue()); // 3.14
  // var stringBox = NumberBox<String>("test"); // Error!
}
✅ Hanya tipe turunan num (int, double) yang diperbolehkan.

🧠 Type Inference (Inferensi Tipe)

Slide 10

Dart bisa menebak tipe generic dari konteks tanpa perlu menuliskannya secara eksplisit.

T myFunc<T>(T value) => value;

void main() {
  var intValue = myFunc(10);      // T diinferensi sebagai int
  print(intValue.runtimeType);    // int

  var stringValue = myFunc("test");
  print(stringValue.runtimeType); // String
}

✅ Manfaat Pembatasan Tipe (Constraints)

Slide 11
  • 🔐 Keamanan tipe lebih baik → mengurangi runtime error
  • 📖 Keterbacaan kode meningkat
  • 🔧 Memungkinkan pemanggilan method dari tipe tertentu
  • 🚫 Mencegah penggunaan tipe data tidak sesuai
  • 🧩 Desain kode lebih terstruktur
void printIterable<T extends Iterable<dynamic>>(T iterable) {
  for (var item in iterable) print(item);
}

void main() {
  printIterable<List<int>>([1,2,3]);    // OK
  printIterable<Set<String>>({'apple','banana'});
  // printIterable<int>(10); // ❌ Error: int bukan Iterable
}

🌟 Contoh Tambahan: Triple<A, B, C>

Bonus
class Triple<A, B, C> {
  A first;
  B second;
  C third;
  Triple(this.first, this.second, this.third);

  @override
  String toString() => '($first, $second, $third)';
}

void main() {
  var triple = Triple<int, String, bool>(1, 'Dart', true);
  print(triple); // (1, Dart, true)
}

Fungsi swap generic:

void swap<T>(List<T> list, int i, int j) {
  T temp = list[i];
  list[i] = list[j];
  list[j] = temp;
}

void main() {
  var list = [1, 2, 3];
  swap(list, 0, 2);
  print(list); // [3, 2, 1]
}
📘 Generic di Dart memberikan fleksibilitas tanpa mengorbankan keamanan tipe.
