🔁 Slide 1: Review Sesi 5 – Class Modifiers sealed | base | final

sealed : hierarki tertutup, pattern matching  |  base : hanya bisa di-extend, tidak di-implement dari luar  |  final : tidak bisa di-extend/di-implement dari luar

sealed class Result {}
class Success extends Result {}
class Failure extends Result {}

Contoh cepat: hierarki tertutup dengan pattern matching.

📐 Slide 5: Sintaks Generic Dasar

• Type parameter: <T>, <E>, <K, V>

• Konvensi:

  • T = Type
  • E = Element (collection)
  • K = Key
  • V = Value
  • R = Return type
class MyClass<T> {
  T data;
  MyClass(this.data);
  T getData() { return data; }
}

T myFunc<T>(T value) { return value; }

void main() {
  // Class generic
  var myIntClass = MyClass<int>(10);
  var myStringClass = MyClass<String>("test");
  print(myIntClass.getData());   // Output: 10
  print(myStringClass.getData());// Output: test
  
  // Fungsi generic
  var intValue = myFunc<int>(100);
  var stringValue = myFunc<String>("hello");
  print(intValue);   // 100
  print(stringValue);// hello
}

📦 Slide 6: Contoh Kelas Box<T>

class Box<T> {
  T? _value;        // Tipe data T, nullable
  Box(this._value); // Constructor
  T? get value => _value;
  void setValue(T value) { _value = value; }
}

void main() {
  // Menggunakan Box dengan tipe data int
  var intBox = Box<int>(10);
  print('Int Box Value: ${intBox.value}'); // Output: 10
  intBox.setValue(20);
  print('Int Box Value After Set: ${intBox.value}'); // 20
  
  // Menggunakan Box dengan tipe data String
  var stringBox = Box<String>("Hello, Generic!");
  print('String Box Value: ${stringBox.value}');
  stringBox.setValue('Hello World');
  print('String Box Value After Set: ${stringBox.value}');
  
  // Box dengan double
  var doubleBox = Box<double>(3.14);
  print('Double Box Value: ${doubleBox.value}'); // 3.14
}

🔗 Slide 7: Multiple Type Parameters

class Pair<K, V> {
  K key;
  V value;
  Pair(this.key, this.value);
  void printPair() { print('Key: $key, Value: $value'); }
}

void main() {
  // Pair dengan tipe data String dan int
  var pair1 = Pair<String, int>('age', 30);
  pair1.printPair(); // Output: Key: age, Value: 30
  
  // Pair dengan tipe data String dan String
  var pair2 = Pair<String, String>('name', 'Alice');
  pair2.printPair(); // Output: Key: name, Value: Alice
  
  // Pair dengan tipe data int dan double
  var pair3 = Pair<int, double>(1, 2.71);
  pair3.printPair(); // Output: Key: 1, Value: 2.71
}

⚙️ Slide 8: Menggabungkan Dua Nilai dengan Tipe Berbeda

String combine<T, U>(T value1, U value2) {

}

🔒 Slide 9: Generic dengan extends (Constraint)

• membatasi tipe data yang dapat digunakan dengan type parameter menggunakan keyword extends.

class NumberBox<T extends num> {
  T value;
  NumberBox(this.value);
  T getValue() { return value; }
}

void main() {
  // Valid
  var intBox = NumberBox<int>(10);
  print("int box value: ${intBox.getValue()}"); // Output: 10
  var doubleBox = NumberBox<double>(3.14);
  print("double box value: ${doubleBox.getValue()}"); // Output: 3.14
  
  // Invalid - Error: The type 'String' is not a subtype of 'num'
  // var stringBox = NumberBox<String>("test");
}

🧠 Slide 10: Type Inference (Inferensi Tipe)

• Dart bisa menebak tipe generic dari konteks

T myFunc<T>(T value) { return value; }

void main() {
  var intValue = myFunc(10);      // Dart menginferensi T sebagai int
  print(intValue.runtimeType);    // Output: int
  var stringValue = myFunc("test"); // Dart menginferensi T sebagai String
  print(stringValue.runtimeType);   // Output: String
}

📌 Slide 11: Pembatasan Tipe dan Manfaatnya

  • Keamanan Tipe yang Lebih Baik: Memastikan kode hanya bekerja dengan tipe data kompatibel, mengurangi runtime error.
  • Peningkatan Keterbacaan Kode: Kode lebih jelas, tipe data yang diharapkan eksplisit.
  • Memungkinkan Penggunaan Method dari Tipe Data Tertentu: Memanggil method spesifik dari tipe yang dibatasi.
  • Mencegah Penggunaan Tipe Data yang Tidak Sesuai: Menghindari penyalahgunaan tipe.
  • Membantu dalam Desain dan Implementasi: Merancang kelas/fungsi generic lebih terstruktur.
void printIterable<T extends Iterable<dynamic>>(T iterable) {
  for (var item in iterable) { print(item); }
}
void main() {
  printIterable<List<int>>([1, 2, 3]); // Output: 1,2,3
  printIterable<Set<String>>({'apple', 'banana'});
  // printIterable<int>(10); // Error: int bukan Iterable
}

🗄️ Generic Cache<T> (Contoh Implementasi)

class Cache<T> {
  final Map<String, T> _storage = {};
  void set(String key, T value) { _storage[key] = value; }
  T? get(String key) { return _storage[key]; }
  void remove(String key) { _storage.remove(key); }
}

void main() {
  var userCache = Cache<String>();
  userCache.set('name', 'Alice');
  print(userCache.get('name')); // Alice
  
  var productCache = Cache<Product>(); // Product class asumsi
  productCache.set('p1', Product('Laptop', 1000));
}

🔄 Bonus: Fungsi swap<T> & Class Triple<A,B,C>

void swap<T>(List<T> list, int i, int j) {
  final temp = list[i];
  list[i] = list[j];
  list[j] = temp;
}

class Triple<A, B, C> {
  A first;
  B second;
  C third;
  Triple(this.first, this.second, this.third);
  void printTriple() => print('Triple($first, $second, $third)');
}

void main() {
  var numbers = [1, 2, 3];
  swap(numbers, 0, 2);
  print(numbers); // [3, 2, 1]
  
  var triple = Triple<int, String, bool>(42, "Dart", true);
  triple.printTriple(); // Triple(42, Dart, true)
}
