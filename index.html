Slide 3 Pengantar Generic

📌 Apa itu Generic? Fitur untuk membuat kode yang bekerja dengan berbagai tipe data.

📦 Analogi: Kotak penyimpanan yang bisa menampung apa saja, tapi dengan label tipe.

✅ Manfaat:
i. Type safety (keamanan tipe)
ii. Code reusability (penggunaan ulang)
iii. Compile-time checking
Slide 4 Masalah Tanpa Generic
class Box {
  dynamic data; // Bisa apa saja
  Box(this.data);
}

void main() {
  var box = Box(42);
  int value = box.data; // Runtime error jika bukan int
  // Tidak aman!
}

⚠️ Tanpa generic, terjadi risiko runtime error karena tipe tidak terkontrol.

Slide 5 Solusi dengan generic
class Box {
  T data;
  Box(this.data);
}

void main() {
  var intBox = Box(42);     // T = int
  var stringBox = Box('Hello'); // T = String

  int value = intBox.data;       // Aman, pasti int
  print(value);                  // Output: 42

  // stringBox.data = 123;       // ❌ ERROR compile-time: String tidak bisa diisi int
  
  // Contoh yang benar mengubah nilai dengan tipe sama
  stringBox.data = 'World';      // ✅ Boleh, karena tipe String
  print(stringBox.data);         // Output: World
}

Slide 6 contoh Sintaks Generic pada class dan function

Type parameter: <T>, <E>, <K, V>

  • T = Type
  • E = Element (collection)
  • K = Key, V = Value, R = Return type
class MyClass<T> {
  T data;
  MyClass(this.data);
  T getData() => data;
}

T myFunc<T>(T value) => value;

void main() {
  var myIntClass = MyClass<int>(10);
  var myStringClass = MyClass<String>("test");
  print(myIntClass.getData());   // 10
  print(myStringClass.getData()); // test

  var intValue = myFunc<int>(100);
  var stringValue = myFunc<String>("hello");
}
Slide 7 Kelas Box<T> - Contoh Praktis
class Box<T> {
  T? _value;
  Box(this._value);
  T? get value => _value;
  void setValue(T value) { _value = value; }
}

void main() {
  var intBox = Box<int>(10);
  print('Int Box Value: ${intBox.value}');
  intBox.setValue(20);
  print('Int Box Value After Set: ${intBox.value}');

  var stringBox = Box<String>("Hello, Generic!");
  print('String Box Value: ${stringBox.value}');
  stringBox.setValue('Hello World');
  print('String Box Value After Set: ${stringBox.value}');

  var doubleBox = Box<double>(3.14);
  print('Double Box Value: ${doubleBox.value}');
}
Slide 8 Multiple type parameters
class Pair {
  K key;
  V value;

  Pair(this.key, this.value);

  void printPair() {
    print('Key: $key, Value: $value');
  }
}

void main() {
  // Pair dengan tipe data String dan int
  var pair1 = Pair("age", 30);
  pair1.printPair(); // Output: Key: age, Value: 30

  // Pair dengan tipe data String dan String
  var pair2 = Pair("name", "Alice");
  pair2.printPair(); // Output: Key: name, Value: Alice

  // Pair dengan tipe data int dan double
  var pair3 = Pair(1, 2.71);
  pair3.printPair(); // Output: Key: 1, Value: 2.71
}

Slide 9 menggabungkan dua nilai dengan tipe data yang berbeda
String combine<T, U>(T value1, U value2) {
  return '$value1, $value2';
}

void main() {
  var result1 = combine<String, int>("Number is ", 10);
  print(result1); // Number is , 10

  var result2 = combine<int, double>(10, 20.5);
  print(result2); // 10, 20.5
}

💡 Menggabungkan dua nilai dengan tipe data berbeda.

Slide 8 Generic dengan extends (Constraint)
class NumberBox<T extends num> {
  T value;
  NumberBox(this.value);
  T getValue() => value;
}

void main() {
  var intBox = NumberBox<int>(10);
  print("int box value: ${intBox.getValue()}");
  var doubleBox = NumberBox<double>(3.14);
  print("double box value: ${doubleBox.getValue()}");
  // var stringBox = NumberBox<String>("test"); // Error: String bukan subtype num
}

🔒 extends membatasi tipe data hanya turunan dari num.

Slide 9 Type Inference (Inferensi Tipe)
T myFunc<T>(T value) => value;

void main() {
  var intValue = myFunc(10); // Dart infer T = int
  print(intValue.runtimeType); // int

  var stringValue = myFunc("test"); // infer String
  print(stringValue.runtimeType); // String
}

🎯 Dart bisa menebak tipe generic dari konteks, sehingga kode lebih ringkas.

Slide 10 Manfaat Pembatasan Tipe (Constraints)
  • ✅ Keamanan Tipe yang Lebih Baik: mengurangi runtime error
  • ✅ Peningkatan Keterbacaan Kode: tipe data eksplisit
  • ✅ Memungkinkan pemanggilan method dari tipe tertentu
  • ✅ Mencegah penggunaan tipe yang tidak sesuai dengan logika
  • ✅ Membantu desain dan implementasi lebih terstruktur
Slide 11 Constraint pada Iterable
void printIterable<T extends Iterable<dynamic>>(T iterable) {
  for (var item in iterable) {
    print(item);
  }
}

void main() {
  printIterable<List<int>>([1, 2, 3]); // 1 2 3
  printIterable<Set<String>>({'apple', 'banana'});
  // printIterable<int>(10); // Error: int bukan Iterable
}

📚 Membatasi tipe parameter agar hanya menerima class yang mengimplementasikan Iterable.

Slide 12 Generic dengan Abstract Class
abstract class Animal {
  void makeSound();
}
class Dog implements Animal {
  @override void makeSound() => print('Woof!');
}
class Cat implements Animal {
  @override void makeSound() => print('Meow!');
}
class AnimalCage<T extends Animal> {
  T animal;
  AnimalCage(this.animal);
  void sound() => animal.makeSound();
}

void main() {
  var dogCage = AnimalCage<Dog>(Dog());
  dogCage.sound(); // Woof!
  var catCage = AnimalCage<Cat>(Cat());
  catCage.sound(); // Meow!
  // var stringCage = AnimalCage<String>("test"); // Error
}

Slide 15 Contoh: Cache Generic
class Cache<T> {
  final Map<String, T> _storage = {};
  void set(String key, T value) { _storage[key] = value; }
  T? get(String key) => _storage[key];
  void remove(String key) => _storage.remove(key);
}

void main() {
  var userCache = Cache<String>();
  userCache.set('name', 'Alice');
  
  // Misal class Product { Product(this.name, this.price); ... }

}
