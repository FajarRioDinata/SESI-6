Pengertian Generic

Generic adalah fitur dalam Dart yang digunakan untuk membuat kode agar dapat bekerja dengan berbagai tipe data secara aman.

Manfaat Generic:
  • Type Safety
  • Code Reusability
  • Compile-time Checking

Contoh Tanpa Generic

class Box {
  dynamic data;

  Box(this.data);
}

void main() {
  var box = Box(42);

  int value = box.data;
  print(value);
}

Penggunaan dynamic kurang aman karena bisa menyebabkan runtime error.

Contoh Dengan Generic

class Box<T> {
  T data;

  Box(this.data);
}

void main() {
  var intBox = Box<int>(42);
  var stringBox = Box<String>("Hello");

  print(intBox.data);
  print(stringBox.data);
}

Dengan Generic, tipe data menjadi lebih aman dan jelas.

Sintaks Generic

  • T = Type
  • E = Element
  • K = Key
  • V = Value
class MyClass<T> {
  T data;

  MyClass(this.data);
}

Multiple Type Parameter

class Pair<K, V> {
  K key;
  V value;

  Pair(this.key, this.value);
}

Generic dapat menggunakan lebih dari satu tipe data.

Generic dengan Extends

class NumberBox<T extends num> {
  T value;

  NumberBox(this.value);
}

Keyword extends digunakan untuk membatasi tipe data tertentu.

Type Inference

T myFunc<T>(T value) {
  return value;
}

void main() {
  var angka = myFunc(10);
  print(angka);
}

Dart dapat menentukan tipe data secara otomatis.

Kesimpulan

  • Generic membuat kode lebih aman
  • Dapat digunakan ulang
  • Mengurangi runtime error
  • Mendukung berbagai tipe data
